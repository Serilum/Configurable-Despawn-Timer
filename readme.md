Despawn Timer is a minimalistic configurable mod which allows items to remain on the ground longer or shorter than the default lifespan. The maximum time an item entity can stay on the floor is&nbsp;2147483647 ticks (=30,000 hours) , which is the highest possible integer value in the programming language Java.<br><br>Minecraft's default is 5 minutes. The mod's default is 10 minutes.</span><br><br>The ItemExpireEvent will still run after the altered lifespan, making it compatible with other mods.<br><br><br><strong><span style="font-size:20px">Configurable:</span> <span style="color:#008000;font-size:14px"><a style="color:#008000" href="https://github.com/ricksouth/serilum-mc-mods/wiki/how-to-configure-mods" rel="nofollow">(&nbsp;how do I configure?&nbsp;)</a></span><br></strong><strong>despawnTimeInTicks</strong>&nbsp;(default = 12000, min 1, max 2147483647): The delay in ticks when an item should despawn, called the lifespan. Minecraft's default time is 6000 ticks. 1 second is 20 ticks.</p>
